# 📝 React Todo App

A simple and clean Todo application built with React.

## 🚀 Features
- ✅ Add, update, delete todos
- 📦 LocalStorage persistence
- 🎨 Responsive modern UI
- ☁️ Light gradient background
- 🔎 Filter tasks by status (optional)
- ⚡ Fast and lightweight

## 📦 Technologies Used
- React
- CSS
- JavaScript
- HTML
